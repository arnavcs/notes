Example Definition Results

Rings

A ring, \((R, +, \times)\), is a set, \(R\), along with two functions, \(+ : R^2 \rightarrow R\) and \(\times : R^2 \rightarrow R\), which we call addition and multiplication. We must have the following ring axioms:

  1. There exists an additive identity.

  2. Every element has an additive inverse.

  3. Addition is commutative.

  4. Addition is associative.

  5. Multiplication is associative.

  6. Multiplication distributes over addition.

We often simply denote this ring by it set of elements, \(R\).

Unital Rings, Units, and the Group of Units

A unital ring is a ring with a multiplicative identity. In a unital ring, \(R\), an element with a multiplicative inverse is called a unit. The group of units of \(R\) is \(R^{\times} := \{ a \in R : a^{- 1} \text{ exists} \}\).

By convention, we say that the trivial ring, \(\{ 0 \}\) is non-unital.

Commutative Rings

A ring, \(R\), is commutative if multiplication commutes, which is to say that \((\forall a, b \in R) (ab = ba)\).

Properties of Rings

From the ring axioms, we get the following results for a ring, \(R\):

  1. The additive inverse of any \(a \in R\) is unique and denoted \(- a\).

  2. The additive identity is unique, and denoted \(0_R\).

  3. \((\forall a \in R) (a \cdot 0_R = 0_R \cdot a = 0_R)\).

  4. \((\forall a, b \in R) ((- a) b = a (- b) = - (ab))\)

And for a unital ring, \(R\), we see:

  1. The multiplicative identity is unique, and denoted \(1_R\).

  2. If \(a \in R^{\times}\), then the multiplicative inverse of \(a\) is unique and denoted \(a^{- 1}\).

Ring Notation

In a ring, \(R\), and for some \(n \in \mathbb{N}\), we generally say that

  1. \(a - b \equiv a + (- b)\)

  2. \(na \equiv a + \overset{n \times}{\ldots} + a \).

  3. If \(R\) is unital, \(n \equiv n 1 \equiv 1 + \overset{n \times}{\ldots} + 1\)

  4. \(a^n \equiv a \times \overset{n \times}{\ldots} \times a\)

Checking Commutativity

Let \(R\) be a ring.

\(\displaystyle (R \text{ commutative}) \Leftrightarrow (\forall a, b \in R) ((a + b)^2 = a^2 + 2 ab + b^2)\)

Binomial Theorem

Let \(R\) be a commutative ring. \((\forall a, b \in R) (\forall n \in \mathbb{N})\)

\(\displaystyle (a + b)^n = \sum_{k = 0}^n \binom{n}{k} a^k b^{n - k}\)

Subring

A subring, \(S\), of \(R\) is a subset of \(R\) which maintains the same ring structure of \(R\). This means it is a ring under the same addition and multiplication definitions for \(R\).

Subring Test

Let \(R\) be a ring, and let \(\varnothing \neq S \subseteq R\).

\(\displaystyle (S \text{ subring of } R) \Leftrightarrow (\forall a, b \in S) (a \times b \in S \wedge a - b \in S)\)

Center of a Ring

Let \(R\) be a ring. The center of the ring is defined by:

\(\displaystyle Z (R) := \{ a \in R : (\forall b \in R) (ab = ba) \}\)

We have that \(Z (R)\) is always a subring of \(R\), and is commutative.

Zero Divisors and Integral Domains

Let \(R\) be a commutative ring. We call \(0_R \neq a \in R\) a zero divisor if \((\exists b \in R, b \neq 0) (ab = 0_R)\). If a ring is commutative, unital, and has no zero divisors, it is an integral domain (or ID for short).

Cancellation

Let \(R\) be an integral domain. \((\forall a, b, c \in R, a \neq 0_R)\)

\(\displaystyle ab = ac \Rightarrow b = c\)

Fields

A field is a commutative unital ring where every non-zero element is a unit.

Polynomial Rings

Let \(R\) be a commutative ring. We say the polynomial ring in \(x\) over \(R\) is the ring of polynomials in \(x\) with coefficients in \(R\). This is to say

\(\displaystyle R [x] := \{ p_0 + p_1 x + \cdots + p_m x^m : m \in \mathbb{N}, p_i \in R, p_m \neq 0_R \operatorname{if}m \neq 0 \}\)

Properties of Fields and Integral Domains

Let \(R\) be a commutative, unital ring. We have the following:

  1. \((R \text{ ID}) \Rightarrow (R [x] \text{ ID})\).

  2. If \(0_R \neq a \in R\) is a zero divisor, then \(a \not\in R^{\times}\).

  3. \((R \text{ field}) \Rightarrow (R \text{ ID})\).

  4. \((R \text{ ID}) \wedge (R \text{ finite}) \Rightarrow (R \text{ field})\).

Image and Preimage

Let \(f : A \rightarrow B\) be a function. And let \(C \subseteq A\), and \(D \subseteq B\). We define the image of \(C\) as \(f (C) := \{ f (x) : x \in C \}\) and we define the preimage of \(D\) as \(f^{- 1} (D) := \{ x \in A : f (x) \in D \}\).

Injective, Surjective, and Bijective

Consider a function \(f : A \rightarrow B\). We say \(f\) is injective (or one-to-one) if

\(\displaystyle (\forall x, y \in A) (f (x) = f (y) \Rightarrow x = y)\)

We say \(f\) is surjective (or onto) if

\(\displaystyle (\forall y \in B) (\exists x \in A) (f (x) = y)\)

And if \(f\) is both surjective and injective, it is bijective (or invertible) and thus has an inverse function, \(f^{- 1}\).

Homomorphisms

Let \(R, S\) be rings. We say that \(\varphi : R \rightarrow S\) is a homomorphism if \((\forall a, b \in R)\)

  1. \(\varphi (a + b) = \varphi (a) + \varphi (b)\)

  2. \(\varphi (ab) = \varphi (a) \varphi (b)\)

Properties of Homomorphisms

Let \(\varphi : R \rightarrow S\) be a homomorphism. Then:

  1. \(\varphi (0_R) = 0_S\)

  2. \((\forall r \in R) (\varphi (- r) = - \varphi (r))\)

  3. \((A \text{ subring of } R) \Rightarrow (\varphi (A) \text{ subring of } S)\)

  4. \((B \text{ subring of } S) \Rightarrow (\varphi^{- 1} (B) \text{ subring of } R)\)

Additionally, if \(\varphi\) is surjective and \(R\) is unital, then:

  1. \(S\) is unital with \(1_S = \varphi (1_R)\)

  2. \((a \in R^{\times}) \Rightarrow (\varphi (a) \in S^{\times} \wedge \varphi (a^{- 1}) = \varphi (a)^{- 1})\)

Kernel

Let \(\varphi : R \rightarrow S\) be a homomorphism. The kernel of \(\varphi\) is

\(\displaystyle \operatorname{Ker} \varphi := \{ x \in R : \varphi (x) = 0_S \} = \varphi^{- 1} (0_S)\)

Then from above, since \(\left\{ {0_S} \right\}\) is a subring of \(S\), we see that \(\operatorname{Ker} \varphi\) is a subring of \(R\).

Embedding

Let \(\varphi : R \rightarrow S\) be a homomorphism. We say that if \(\varphi\) is injective, then it is an embedding. Additionally, we can determine if \(\varphi\) is an embedding since

\(\displaystyle (\varphi \text{ embedding}) \Leftrightarrow (\operatorname{Ker} \varphi = \{ 0_R \})\)

Isomorphism and Isomorphic

Let \(\varphi : R \rightarrow S\) be a homomorphism. If \(\varphi\) is bijective, we call it an isomorphism. Additionally, we then say that \(R\) is isomorphic to \(S\), denoted \(R \cong S\).

Ideal and Proper Ideal

Let \(R\) be a ring. We say \(I\) is an ideal of \(R\) if it is a subring of \(R\) and \((\forall r \in R) (\forall a \in I) (ar, ra \in I)\). If \(I\) is an ideal of \(R\) and \(I \neq R\), then \(I\) is a proper ideal of \(R\).

Ideal Test

Let \(R\) be a ring, and let \(\varnothing \neq I \subseteq R\).

\(\displaystyle (I \text{ ideal}) \Leftrightarrow (\forall a, b \in I) (\forall r \in R) (a - b, ar, ra \in I)\)

Principal Ideal

Let \(R\) be a commutative, unital ring, and let \(r \in R\) be fixed. The principal ideal generated by \(r\) is \(\langle r \rangle := \{ rx : x \in R \}\).

Equivalence Relations

An equivalence relation is a relation that is reflexive, symmetric, and transitive.

Modular Arithmetic, Equivalence Classes, and Quotient Rings

Let \(I\) be an ideal of the ring \(R\). We define an equivalence relation, \(\sim\), such that \((\forall a, b \in R) (a \sim b \Leftrightarrow b - a \in I)\). We will use the notation \(a \equiv b (\operatorname{mod}I)\) to mean \(a \sim b\).

For some \(a \in R\), the equivalence class of \(a\) in \(R\) is \([a]_I := \{ x \in R : a \equiv x (\operatorname{mod}I) \}\). Notationally, \(\bar{a}\) is the same as \([a]_I\).

Now define \(R\) mod \(I\) as \({}^R /_I := \{ [a]_I : a \in R \}\) with operations defined so that \((\forall a, b \in R) ([a]_I + [b]_I = [a + b]_I \wedge [a]_I \cdot [b]_I = [a \cdot b]_I)\). This is a what we call a quotient ring.

Modular Arithmetic with \(\mathbb{Z}\)

For \(n \in \mathbb{Z}^+\), we define \(\mathbb{Z}_n := {}^{\mathbb{Z}} /_{\langle n \rangle} = \{ \bar{0}, \bar{1}, \ldots, \overline{n - 1} \}\), and by convention we write \((\operatorname{mod}n)\) to mean \((\operatorname{mod} \langle n \rangle)\).

Fermat's Little Theorem

Let \(p\) be prime, then \((\forall a \in \mathbb{Z}_p, a \neq 0_{\mathbb{Z}_p})\)

\(\displaystyle a^{p - 1} = 1_{\mathbb{Z}_p}\)

Prime and Maximal Ideals

With reference to prime and maximal ideals of a ring \(R\), we assume that \(R\) is a commutative, unital ring. We say an ideal, \(I\), of \(R\), is prime if

  1. \(I\) is a proper ideal of \(R\)

  2. \((\forall a, b \in R) ((ab \in I) \Rightarrow (a, b \in I))\)

Additionally, we say that \(I\) is maximal if

  1. \(I\) is a proper ideal of \(R\)

  2. \((\forall \text{ ideals } J \text{ of } R) ((I \subsetneq J \subseteq R) \Rightarrow (J = R))\)

Prime and Maximal Ideal Quotient Rings

Let \(I\) be an ideal of ring \(R\). We have the following two results:

  1. \((I \text{ prime}) \Leftrightarrow \left( {}^R /_I \text{ ID} \right)\)

  2. \((I \text{ maximal}) \Leftrightarrow \left( {}^R /_I \text{ field} \right)\)

And since \(\left( {}^R /_I \text{ field} \right) \Rightarrow \left( {}^R /_I \text{ ID} \right)\), we get as a corollory that \((I \text{ maximal}) \Leftrightarrow \left( {}^R /_I \text{ field} \right) \Rightarrow \left( {}^R /_I \text{ ID} \right) \Leftrightarrow (I \text{ prime})\).

Corollory of Zorn's Lemma

Let \(R\) be a commutative, unital ring. \((\forall I \text{ proper ideal of } R) (\exists M \text{ maximal ideal of } R) (I \subseteq M)\).

Additionally, if \(a \not\in R^{\times}\), then \((\exists M \text{ maximal ideal of } R) (a \in M)\).

Existence of a homomorphsim

Let \(I\) be an ideal of ring \(R\). Then, there is some ring \(S\) and homomorphism \(\varphi : R \rightarrow S\) such that \(I =\operatorname{Ker} \varphi\).

First Isomorphism Theorem

Let \(\varphi : R \rightarrow S\) be a ring homomorphism between rings \(R\) and \(S\). Now,

\(\displaystyle {}^R /_{\operatorname{Ker} \varphi} \cong \varphi (R)\)

Ideal Sum

Let \(I, J\) be ideals of \(R\). We define the ideal sum \(I + J\) as follows:

\(\displaystyle I + J := \{ a + b : a \in I, b \in J \}\)

Comaximal Ideals

Ideals \(I, J\) of \(R\) are comaximal if \(I + J = R\)

Ideal Product

Let \(I, J\) be ideals of \(R\). The ideal product of \(I\) and \(J\) is defined as:

\(\displaystyle IJ := \{ a_1 b_1 + \cdots + a_n b_n : n \in \mathbb{N}, a_i \in I, a_j \in J \}\)

If we have multiple ideals \(I_1, I_2, \ldots, I_m\), then we can simplify the product to get

\(\displaystyle I_1 I_2 \cdots I_m = \{ a_{1 1} \cdots a_{1 m} + \cdots + a_{n 1} \cdots a_{n m} : n \in \mathbb{N}, a_{i j} \in I_j \}\)

Property of Comaximality

Let \(I, J\) be comaximal ideals of the commutative, unital ring \(R\). We then have that \(IJ = I \cap J\).

Chinese Remainder Theorem

Let \(R\) be a commutative, unital ring with comaximal ideals \(I\) and \(J\). We have that

\(\displaystyle {}^R /_{IJ} \cong {}^R /_I \times {}^R /_J\)

Where \(\times\) is the cartesian product.
