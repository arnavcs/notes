1Introduction

What is Logic?

Logic is not about what is true or false, but rather the creation of conclusions from a set of assumptions.

Socrate's Logic

This is an example of valid logic:

If all men are mortal Assumption 1
and
Socrates is a man Assumption 2
then
Socrates is mortal Conclusion

Russell's Paradox

Russell's Paradox involves the following scenario. 

Assume there is a village in which no man grows a beard. 
There is only one barber in that village. 
Every morning, this barber shaves every (and only every) man who does not shave thmeselves. 
Does the barber shave themselves?

the answer to the paradox is that such a village cannot exist.

The Berry Paradox 

Which numbers can be defined by an English sentence of at most 200 characters?

2Propositional Logic

Syntax vs. Semantics

Logic is based on the distinction between syntax (the form) and semantics (the meaning).

How do we define sets?

Let's take a look at how to ch aracterize or describe the language of propositional language. We are wondering if there is a set which describes our language. There are the following ways to define sets:

  1. Explicitly list the members of the set

  2. Describe a common property of all the elements of the set (a simple membership condition)

  3. Have an inductive membership condition.

Operations

For a set \(U\), operations over \(U\) are functions of type \(U^k \rightarrow U\) for finite \(k\). For some subset \(X \subseteq U\), we say that \(X\) is closed under an operation \(f : U^k \rightarrow U\) if \((\forall x_1, \ldots, x_k \in X) (f (x_1, \ldots, x_k) \in X)\).

We can write an operation \(f : U^k \rightarrow U\) in the format \(\frac{\alpha_1, \alpha_2, \ldots, \alpha_k}{f (\alpha_1, \alpha_2, \ldots, \alpha)}\).

Minimal Set

Let \(\mathcal{F}\) be a collection of sets. We say that \(A \in \mathcal{F}\) is the minimal set in \(\mathcal{F}\) if \((\forall B \in \mathcal{F}) (A \subseteq B)\).

Inductive Set

Given a universe \(U\), a core set \(C \subseteq U\), and operations \(O\), we define the inductive set, \(I (U, C, O)\) as the minimal set in \(\mathcal{F}= \{ A : (C \subseteq A \subseteq U) \wedge (A \text{ closed under } O) \}\). This is well defined because we have that \((A, B \in \mathcal{F}) \Rightarrow (A \cap B \in \mathcal{F})\). Thus, we see that \(I (U, C, O) = \bigcap \mathcal{F}\).

Words of Propositional Logic

Let \(L\) be the set of all lower case letters with or without indicies. Let \(\Sigma := \left\{ \neg, \wedge, \vee, \text{\text{{\ttfamily{(}}}}, \text{\text{{\ttfamily{)}}}} \right\} \cup L\) and let \(\Sigma^{\ast}\) be the set of all finite sequences of elements of \(\Sigma\).

If \(U = \Sigma^{\ast}\), \(C = L\), and \(O = \left\{ \frac{\alpha, \beta}{(\alpha \vee \beta)}, \frac{\alpha, \beta}{(\alpha \wedge \beta)}, \frac{\alpha, \beta}{(\alpha \rightarrow \beta)}, \frac{\alpha}{(\neg \alpha)} \right\}\), then the words of propositional logic are \(I (U, C, O)\).

Generating Sequence

We say that for an inductive set \(I (U, C, O)\), if we want to show membership of \(\alpha\), then we define a generating sequence. \(\alpha_1, \ldots, \alpha_n\) is a generating sequence for \(\alpha\) iff

  1. \(\forall i \leq n\) either \(\alpha_i \in C\), or \((\exists f \in O, f : U^k \rightarrow U) (\exists j_1, \ldots, j_k < i) (f (\alpha_{j_1}, \alpha_{j_2}, \ldots, \alpha_{j_k}) = \alpha_i)\).

  2. \(\alpha_n = \alpha\).

We have that \(a \in U\) iff there exists a generating sequence for \(\alpha\) with respect to \(U, C, O\).

How do we show Something isn't a Proposition?

We use structural induction to prove that something isn't in \(I (U, C, O)\). We show the property is satisfies by all elements of the core set, and show that the operations maintain the property. Then, we show that the object which we are considering does not have this property.

Structural Induction

Structural induction involves a property which we show is invariant throughout the structure. We use the structure of the construction to show this property is true of anything in in the structure.

Principle of Structural Induction for Inductive Sets

For an inductive set \(I (U, C, O)\), if we have a property, let \(P \subseteq U\) be the subset of \(U\) which satisfies the property. If we have

  1. \(C \subseteq P\)

  2. \((\forall f \in O) (P \text{ is closed under } f)\)

then we have that \(I (U, C, O) \subseteq P\) which means that the property is true for all members of the inductive set.

Proper Initial Segment

Given a finite sequence of symbols \(\alpha = a_1, a_2, \ldots, a_n\), we define a proper initial segment of \(\alpha\) to be the segment \(a_1, a_2, \ldots, a_k\) for some \(1 \leq k < n\).

Properties of Propositional Logic

We can use structural induction to determine the following for the words of propositional logic \(P := I (U, C, O)\). Let \(\#_l, \#_r : U \rightarrow \mathbb{N}\) be the number of left and right brackets in the given word.

  1. \((\forall \alpha \in P) (\alpha \in C \vee \alpha \text{ has brackets})\)

  2. \((\forall \alpha \in P) \left( \alpha \in C \vee \alpha \text{ starts with \text{{\ttfamily{(}}}} \right)\)

  3. \((\forall \alpha \in P) (\#_l (\alpha) =\#_r (\alpha))\)

  4. \((\forall \alpha \in P) (\forall \alpha' \text{ proper initial segmentation of } \alpha) (\#_l (\alpha) >\#_r (\alpha))\)
